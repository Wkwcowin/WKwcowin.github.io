## 修复代码块高亮行横向滚动背景不全问题

**问题**: `docs/linktech.md` 中代码块使用 `hl_lines` 高亮时，横向滚动后右侧高亮背景消失。高亮只覆盖文字部分，右侧空白区域没有背景。

**根本原因**: 主题中 `code` 使用 `display: block`，其宽度等于 `pre` 容器宽度。当代码内容超出容器宽度产生横向滚动时，`.hll`（`display: block`）的宽度被限制在容器宽度内，无法延伸到滚动区域。

**最终修复方案**（`docs/stylesheets/extra2.css`）：
```css
/* pre 负责滚动 */
.md-typeset .highlight pre,
.md-typeset pre {
  overflow-x: auto;
}

/* code 宽度等于内容实际宽度 */
.md-typeset pre > code {
  display: inline-block !important;
  min-width: 100%;
  vertical-align: top;
}

/* hll 作为 block 填满 code 宽度，背景覆盖整行 */
.md-typeset .highlight .hll,
.md-typeset pre code .hll {
  display: block !important;
}
```

**原理**: 将 `code` 设为 `inline-block`，其宽度自动等于内容实际宽度（而非容器宽度）。`.hll` 保持 `display: block`，自动填满 `code` 的完整宽度，从而使高亮背景延伸到滚动区域。

**修改文件**: `docs/stylesheets/extra2.css`
